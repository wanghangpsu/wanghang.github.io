Image Privacy Classification

Used EM algorithm to train the model by iteratively update the latent (group) and weight parameters
Processed each image into 100 overlapping patches, applied a generalized logistic regression function to get the probability of a patch being private, and put the top 10 sorted probabilities into a softmax layer (logistic function) to get the probability of an image being private
Extended the model to include the profile data of every user to define the group membership of the user
Introduced ensemble learning method, divided the training set into 5 sets, trained them separately and then used all the five model to decide the image privacy by doing soft vote based on the probability of being private
Got 86% accuracy with 94.2% TNR and 64.5% TPR, which is better than published previous work on the same dataset (overall 79% accuracy), with the ROC curve equivalent to previous work

Image Classification on Oxford-IIIT Pet Dataset Based on Transfer Learning

Used a pre-trained eight-layer implementation of Convolutional Neural Networks, disregarded the 8th layer, and treated the 4096-dimensional output of the 7th layer of the network as a feature vector
Implemented a linear SVM as binary classifier (Cat and Dog) and multi-class classifier (different kinds of cats and dogs), also implemented a generalized logistic function classifier as multi-class classifier
Got 98.5% accuracy for the binary classification and 89.3% accuracy for the multi-class (37 class) classification
Got 75% accuracy for the cat and dog classification for kinds of cats or dogs not in the Pet Dataset

Transient Heat Conduction Equivalence Model of Modular Multi-level Converter in Ultra High Voltage Direct Current

Investigated the Junction temperature changes of Insulated Gate Bipolar Thyristor (IGBT) module in (modular multi-level converter) MMC in Ultra High Voltage Direct Current (UHVDC) system Using Ansys simulation
Simulated thermal resistance curves between the IGBT module junction under different operating conditions
Concluded that the change of junction temperature only affects the thermal coefficient of the chip layer, and it does not dramatically affect the transient thermal conductivity equivalent model of the IGBT module
